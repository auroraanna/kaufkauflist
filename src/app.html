Manual

kaufkauflist is a web app for creating and managing to-do lists for shopping or other use cases.

When an item is checked, it means that that item has been bought.

You can share a list with others by just giving them the current URL when viewing a list. Make sure to not leave out the query parameters (everything after ?). The query parameters are used for authentication.

For buttons that are next to the text of an item, the ones with the ✏️ emoji let you edit the item's name and the ones with the ❌ emoji delete that item.
